# STM32F103 ADC Simulation Test

This repository contains an embedded C firmware application developed for the STM32F103C8T6 (Blue Pill) microcontroller. The project focuses on configuring and testing the internal Analog-to-Digital Converter (ADC) peripheral using the STM32 CubeHAL library.

## Features
- **ADC Configuration:** Configured to sample analog voltage inputs continuously.
- **Hardware Simulation:** Fully tested inside the PICSimLab simulation environment using the virtual Blue Pill board.

## Tools Used
- **STM32CubeIDE**: Firmware development, compilation, and code generation.
- **PICSimLab**: Simulating the hardware behavior and validating binary execution.
